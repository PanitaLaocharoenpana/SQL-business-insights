# SQL Business Insights

## Project Overview

This project demonstrates SQL techniques used to answer business questions from transactional sales data.

The analysis includes:

- Basic filtering
- Aggregations
- Subqueries
- INNER JOIN
- LEFT JOIN
- Multi-table analysis

## Business Questions

### Product Analysis

- Which products contain specific keywords?
- Which products have the highest prices?

### Sales Analysis

- How often do specific prices appear?
- Which products have the highest quantities sold?

### Customer Analysis

- Which customers purchased specific products?
- Which customers placed orders containing premium items?

## SQL Skills Demonstrated

- SELECT
- WHERE
- ORDER BY
- LIKE
- COUNT()
- SUM()
- GROUP BY
- Subqueries
- INNER JOIN
- LEFT OUTER JOIN
